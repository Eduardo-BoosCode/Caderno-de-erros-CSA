# 🧠 Caderno de Erros PRO+ (ENEM)

Plataforma inteligente para análise de simulados do ENEM.

## 🚀 Funcionalidades

- 📊 Cálculo automático de acertos e percentual
- 📈 Gráfico de desempenho por matéria
- 🧠 Diagnóstico inteligente de estudo
- 🎯 Identificação da matéria prioritária
- 📥 Geração de relatório em Word
- 📊 Exportação de dados

---

## 📱 Como usar

1. Acesse o site do projeto
2. Insira:
   - Gabarito
   - Suas respostas
   - Matérias (opcional)
3. Clique em **Gerar análise**
4. Veja:
   - Seu desempenho
   - Gráficos
   - Recomendações de estudo
5. Baixe o relatório

---

## ⚙️ Tecnologias utilizadas

- Python
- Streamlit
- Pandas
- Matplotlib
- Python-docx

Ajudar estudantes a identificar erros com precisão e melhorar o desempenho no ENEM através de análise estratégica.
